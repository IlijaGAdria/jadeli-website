# Case Couture Starter

Fashion-forward Next.js starter inspired by editorial phone-case stores, adapted into an original landing page for a shop that only sells cases for the latest two iPhone and Galaxy generations.

## Stack
- Next.js App Router
- React
- TypeScript
- CSS Modules
- npm workspaces monorepo

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Included
- Announcement bar
- Editorial hero
- New drops grid
- Brand/device selector section
- Ready-to-extend monorepo structure
